## Technology Stack

```bash
- TypeScript
- React.js 18
- Next.js 13
- Redux for State Management
- TailwindCSS 3.3.2
```

## Getting Started

First, run the development server:

```bash
$ npm run dev
# or
$ yarn dev
# or
$ pnpm dev
```

For build to production run:
```bash
$ npm run build
```

For run Unit Testing:
```bash
$ npm run test:unit
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
